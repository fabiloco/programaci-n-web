Metodología de BEM

En CSS existen muchas metodologías para trabajar de forma organizada y llevar un orden al momento de diseñar nuestras páginas webs. Una de estas metodologías es la Metodología de BEM. Esta metodología nos ayuda organizar nuestros bloques y a llevar un orden herarquico que sea facil de llevar, buscar, entender y leer.

Para trabajar con la metodología de BEM debemos colocar a nuestras etiquetas clases. Los nombres de esas clases deben ser nombres claves, cortos que describan facilmente que es lo que va a ser ese bloque. Luego de eso, todos los elementos que esten dentro de ese bloque, también tendran una clase, cuyo nombre empezara con el nombre de la clase del bloque, seguido de dos underscore ( __ barras bajas), y luego el nombre del elemento. y Así sucesivamente. Ejemplo:


<p class="parrafo" > <b class="parrafo__bold" >Gatitos<b> y <i class="parrafo__italic >cafe<i> </p>

Luego, en CSS, para modificar estos elementos, se vería así:

.parrafo{
font-family:Tahoma;
}

.parrafo__bold{
color:red;
}

.parrafo__italic{
color:blue;
}
