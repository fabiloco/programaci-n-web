Selectores de CSS

Existen muchas formas distintas en las que se puede modificar y dar estilo a un documento de HTML. Podemos hacerlo desde el propio documento HTML con una etiqueta (<style>) o incluso también se puede hacer importando en el documento HTML un archivo propio con formato .css el cual contendra todas las propiedades de estilo de los elementos presentes en el documento HTML.

Esta ultima opción resulta ser muchas veces la más eficiente y recomendada para usar, ya que mantenemos el HTML de un lado y el CSS del otro, sin juntar lengajes.

A continuación, repasaremos todos los tipos de selectores que hay, como implementarlos y para que sirven:

Selector universal

El selector universal selecciona a todos los elementos que hay dentro del body. Con este selector podemos, por ejemplo, modificar la fuente de toda la página ene general, o todo lo que esta dentro del body.

Para usarlo, indicamos como selector el signo asterisco (*), y de esa forma ya le indicamos a CSS que le queremos modificar una propiedad a todos los elementos de la página.

*{
propiedad:valor;
propiedad:valor;
}

Selector de tipo

Con este selector, seleccionamos a los elementos por tipo. Es decir, simplemente con su nombre de etiqueta, y una vez seleccionado, todos los elementos que sean de ese tipo, serán modificados.

Para usar el selector por tipo, simplemente debemos indicar en la parte del selector el nombre de la etiqueta a la cual queremos modificar, de la siguiente forma:

h1{
propiedad:valor;
propiedad:valor;
}

Selector por clases

Con el selector por clases, podremos modificar todos los elementos que pertenezcan a la misma clase. Para atribuirle una clase a un elemento de HTML, simplemente debemos añadirle a la atiqueta el atributo class y darle un nombre a la clase.

Por ejemplo, vamos añadirle la clase pepito a un elemento h2:

<h2 class="pepito" > Titulo </h2>

Ahora, para hacer hacer referencia a la clase pepito en CSS, debemos usar el selector de clase, que es simplemente colocar el nombre de la clase, pero empezando por un punto. .pepito

Veamos el ejemplo si queremos modificar cualquier elemento de la clase pepito:

.pepito{
propiedad:valor;
propiedad:valor;
}

Selector por id (identificador)

El selector por identificador es muy parecido al selector por clases, ya que de igual manera, debemos añadir un atributo, esta vez de tipo id , al elemento que queramos modificar, y todos los elementos que tengan este identificador especial, serán modificados.

Por ejemplo, vamos a añadirle el id=id-element a la etiqueta h2:

<h2 id="id-element" > Titulo </h2>

* Algo muy importante de los identificadores, es que estos deben ser unicos. Un identificador debe ser unico de un elemento en concreto. Ningun otro elemento particular debe tener el mismo id que otro.

Ahora bien, para usar el selector por identificador en CSS, debemos usar el # hashtag (o el gato XD) y luego colocar el valor del id que pusimos en la etiqueta que queremos modificar.

#id-element{
propiedad:valor;
propiedad:valor;
}

Selector por atributo

El selector por atributo trata sobre colocarle un atributo y un valor al elemento que queramos modificar, y luego en CSS modificar todos los elementos que tengan el mismo atributo igualados al mismo valor. Para añadir un atributo a una etiqueta HTML, debemos escribir una propiedad y igualarla a un valor, y estos dos items pueden ser los que sean, como en el siguiente ejemplo:

<h2 love="cats" > Titulo </h2>

Luego, en nuestro archivo CSS, para usar el selctor por atributo, debemos escribir lo mismo que escribimos en nuestro elemento HTML tal cual como esta, encerrado entre corchetes. [bebesita="anuel"].

[love="cats"]{
propiedad:valor;
propiedad:valor;
}

Selector por decendientes

La forma de seleccionar de forma decendente es simplemente seleccionar los elementos que esten dentro de otros. Primero se pone el elemento en el cual esta contenido el elemento que queremos modificar, y luego se pone el elemento que queremos modificar que este adentro de ese primer elemento contenedor. Por ejemplo, supongamos que tenemos las siguientes etiquetas:

<p> <b>Gatitos<b> y <i>cafe<i> </p>

Ahora, si queremos modificar esos dos elementos que estan dentro de la etiqueta parrafo con el selector decendente, podemos hacer lo siguiente:

p b{
color:red;
}
p i{
color:blue;
}

De esta forma, se modificaran esos dos elementos que estan dentro de la etiqueta parrafo, con los respectivos colores que le asignamos a cada elemento. Esto también debería funcionar perfectamente con elementos que esten dentro de una clase, mezclando así los tipos de selectores.

Seleccion por pseudo-clases

La forma de seleccionar por pseudo clases es para darle funcionalidad de eventos a los elementos. Es decir, cuando se active un cierto evento en el documento HTML, la pseudo clase se activará y entonces el elemento cambiara sus propiedades a las que le hayamos puesto. Estas pseudo clases ya estan definidas, algunas muy comunes son hover, que se activa cuando pasamos el puntero sobre el elemento.

Veamos en que consiste, podemos hacer que un elemento cambie de color cuando pasemos el puntero sobre el con la pseudo clase hover, para hacerlo simplemente debemos seleccionar el elemento con algún tipo de selector, y luego de eso, poner dos puntos (:), seguido de la pseudo clase que queramos utilizar.

button:hover{
color:red;
}
