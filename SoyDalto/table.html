Tablas en HTML

Las tablas en HTML se administran con la etiqueta <table>. Sirven para guardar y organizar información. Podemos
almacenar esta información dentro de los dos elementos de las tablas; las columnas y las filas. Estos elementos los
encontramos representados por las siguientes etiquetas; <tr> que es para definir las filas y <td> que es para definir
las columnas.

Las columnas se usan para almacenar información del mismo tipo, mientras que las filas se usan para cambiar el tipo de información.
