Etiquetas de multimedia en HMTL

En HTML existen multiples etiquetas etiquetas para introducir objetos multimedia en una página.
De la misma manera que ocurre con los enlaces, estos elementos multimedia pueden ser locales o
externos. Pueden estar en las carpetas del servidor o pueden provenir de internet, eso depende
de la ruta que le demos al parametro src, el cuál esta presente en todos las siguientes
etiquetas. También podemos usar las etiquetas height y width para modificar el tamaño a gusto.

Etiqueta para imagenes

Tenemos la etiqueta <img> que es para insertar imagenes.

Etiqueta para videos

Tenemos la etiqueta <video> que es para insertar videos.

Es importante que cada vez que se use esta etiqueta, se debe agregar dentro de los parametros
el parametro controls que representan los controles para poder reproducir, pausar y volumen.

Etiqueta para audios

Tenemos la etiqueta <audio> que es para insertar audios.

Es importante que cada vez que se use esta etiqueta, se debe agregar dentro de los parametros
el parametro controls que representan los controles para poder reproducir, pausar y volumen.

Modificar el icono de la página

Para modificar el icono del documento HTML en el que estamos trabajando, debemos usar una etiqueta
dentro de la etiqueta "head". Esta etiqueta es la etiqueta link. Para que la etiqueta link reconozca
que queremos usarla para cambiar el icono del documento, tenemos que poner el parametro rel y darle
"icon" como propiedad, seguido de eso, en el parametro hrel, debemos darle la ruta de la imagen, la cual
debe estar en formato ".ico". Haciendo todo esto, la etiqueta completa deberia quedar de la siguiente manera

<link rel="icon" hrel="icono.ico">
