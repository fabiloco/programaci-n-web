Formularios en HTML

Los formularios en HTML son un conjunto de elementos dinamicos que sirven para interactuar con el usuario. Son
más que todo elementos de de entrada para que el usuario digite elementos y se puedan mandar al lado del servidor.
Elementos como campos de texto, combobox, radiobuttons, entre muchos otros, son los que se
pueden utilizar.

Primero que todo, se debe usar la etiqueta <form> para indicar que se esta iniciando un formulario.

Etiqueta <input>

Esta etiqueta sirve para capturar datos que el usuario digite en un campo de texto.

Debemos indicarle tres atributos. El tipo de input que será, el nombre que tendrá la variable (es algo que será
mucho más util de parte del servidor "backend") y además el valor inicial que queramos asignarle al input.

Hay distintos tipos de input. Veamos algunos de ellos.

type=text (por defecto)

type=number (spinner)

type=color

type=range

type=url

type=tel

Algunos atributos para los inputs

Algunos de los atributos que podemos usar en la etiqueta input, además de los que ya vimos,
son los siguientes:

  • required: este atributo indica que el elemento input debe ser llenado
    obligatoriamente.

Algunos otros elementos etiquetas:

<textarea>

Agrega un cuadrado de texto que se puede agrandar o achicar.

atributo:readonly

Sirve para que el elemento se convierta en solo lectura.
