Metadatos en HTML

Los metadatos son información que sirve para describir otra información u objetos. En un
documento HTML, los metadatos sirven para describir información del propio documento, de su
contenido o de sus objetos.

Estos metadatos se administran con la etiqueta <meta>. Los datos que podemos
administrar en esta etiqueta dependen del atributo que le demos.

Un metadato muy común y necesario para los hispano hablantes es el metadato charset. con
este metadato podemos cambiar la codificación del texto de nuestro documento, que en defecto viene
en ingles, lo cual no nos permite utilizar acentos y caracteres con tilde. Para solucionar esto,
simplemente tenemos que usar la etiqueta y ponerle como atributo charset=utf-8, la cual es la
codificación para que se puedan visualizar los caracteres con tilde.

<meta charset=utf-8>

Es muy importante tener esta etiqueta meta en todas los documentos de HTML

Ahora, los otros atributos que tiene meta son name y content. Estos sirven para agregar
otros tipos de metadatos. por ejemplo:

name=keywords, palabras clave o etiquetas

Este tipo de metadato sirve para posicionar la página web en el buscador. Ahora mismo no son tan
relevantes como alguna vez lo fueron, pero sin embargo sigue sirviendo y ayudando un poco al posicionamiento.

<meta name="keywords" content="programación,web,HTML,CSS,Javascript">

name=description, descripción de la página

Con el metadato description se describe la página, su contenido, que es, para qué es.

<meta name="description" content="Esta es una página para documentar y apuntar mi proceso de aprendizaje con HTML, para poder recordarlo y consultarlo cada vez que sea necesario, además de poder practicar al ritmo.">

name=author, nombre del autor de la página

Simplemente guarda el dato del nombre del autor de la página.

<meta name="author" content="Fabián Sánchez">

name=copyright, derechos de autor

Sirve para dar los derechos de autor a la página, en caso de que trabajemos para una empresa, por ejemplo:

<meta name="copyright" content="Facebook inc.">

name=robots, página indexable o no

Esta página sirve para indicar si la página es indexable o no, es decir, si puede salir en las busquedas de
navegadores como google o no puede salir. Por ejemplo:

Si se quiere que la página sea indexable

<meta name="robots" content="index">

Si se quiere que la página NO sea indexable

<meta name="robots" content="noindex">
